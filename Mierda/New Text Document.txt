# Escena 1: Definición de sección cónica degenerada

- Mostrar texto animado:
  "Definición: Se denomina sección cónica degenerada a la intersección de un cono circular recto de dos hojas con un plano que pasa por su vértice. Se clasifican en tres tipos: punto, recta y par de rectas."
- Animar la aparición del texto con Write o FadeIn.
- Mostrar un cono circular recto de dos hojas (objeto 3D o 2D simplificado).
- Mostrar un plano que pasa por el vértice del cono.
- Animar la intersección para mostrar:
  - Caso 1: Sección puntual (resaltar el punto de intersección).
  - Caso 2: Sección recta (mostrar una línea de intersección).
  - Caso 3: Sección par de rectas (mostrar dos líneas intersectándose).
- Etiquetar cada tipo con texto pequeño y colores distintos.
- Pausa con wait para asimilar.
- Transición suave (FadeOut o FadeTransform) para cerrar la escena.

# Escena 2: Definición de traslación de un sistema de coordenadas

- Mostrar texto animado:
  "Definición: Se denomina traslación de un sistema de coordenadas 𝑂𝑋𝑌 en el plano, a otro sistema de coordenadas 𝑂′𝑋′𝑌′ tal que los ejes 𝑂′𝑋′ y 𝑂′𝑌′ son paralelos a los ejes 𝑂𝑋 y 𝑂𝑌, respectivamente, y además, tienen el mismo sentido. El vector 𝑂𝑂′ se denomina vector de traslación."
- Animar la aparición del texto con Write o FadeIn.
- Mostrar sistema de coordenadas original (Axes con ejes OX y OY).
- Mostrar sistema trasladado (Axes desplazados) paralelo y con mismo sentido.
- Dibujar y animar el vector de traslación \(\overrightarrow{OO'}\) con una flecha y etiqueta.
- Animar la transición del sistema original al trasladado para visualizar el movimiento.
- Pausa con wait para asimilar.
- FadeOut para cerrar la escena.
